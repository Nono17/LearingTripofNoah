### Task1（8.3-8.5）
#### 1. 数组
1. 实现一个支持动态扩容的数组，支持增删改操作
2. 实现两个有序数组合并为一个有序数组
3. 学习哈希表思想，并完成leetcode上的两数之和(1)及Happy Number(202)！

3. 实现两个有序的链表合并为一个有序链表
4. 实现求链表的中间结点
